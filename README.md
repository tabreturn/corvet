corvet
======
An automatic marker for vector graphics drawing tasks.

Run
---
CD to the project root and run:
* ./node_modules/.bin/browser-sync start --config cfg/bs-config.js
* sass --watch scss:css
* source venv/bin/activate; python py/http_api.py
